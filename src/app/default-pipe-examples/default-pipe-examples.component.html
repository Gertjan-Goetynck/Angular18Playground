Default pipes

Source

Angular supplies us with some default pipes for easy data formatting or manipulation.

Some of these pipes are influenced by the app locale. For these settings, look at app.config.ts and main.ts

Date formatting example: {{ '2024-12-31'| date }}

Currency formatting example: {{ 1000.123 | currency }}

Decimal formatting example: {{ 100.123556 | number }}

Percentage formatting example: {{ 0.33 | percent }}

Async pipes can be used to subscribe to an observable. Click the button to update the value inside of it and see this in action.
